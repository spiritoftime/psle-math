## Cube and Cuboid Formulas

### Cube

| Property                | Formula | Variables              |
| ----------------------- | ------- | ---------------------- |
| Edge Length             | a       | a = length of one edge |
| Volume                  | V = a³  | a = edge length        |
| Perimeter (of one face) | P = 4a  | a = edge length        |

### Cuboid (Rectangular Prism)

| Property            | Formula       | Variables                         |
| ------------------- | ------------- | --------------------------------- |
| Dimensions          | l, w, h       | l = length, w = width, h = height |
| Volume              | V = l × w × h | l = length, w = width, h = height |
| Perimeter (of base) | P = 2(l + w)  | l = length, w = width             |

## Common Unit Conversions

### Volume

| Conversion                  | Formula        |
| --------------------------- | -------------- |
| Cubic centimeters to liters | 1000 cm³ = 1 L |
| Liters to milliliters       | 1 L = 1000 mL  |
| Cubic meters to liters      | 1 m³ = 1000 L  |
