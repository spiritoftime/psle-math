## 1. Geometric Shape Formulas

### 1.1 Square

| Property  | Formula | Variables       |
| --------- | ------- | --------------- |
| Area      | A = s²  | s = side length |
| Perimeter | P = 4s  | s = side length |

### 1.2 Rectangle

| Property  | Formula      | Variables             |
| --------- | ------------ | --------------------- |
| Area      | A = l × w    | l = length, w = width |
| Perimeter | P = 2(l + w) | l = length, w = width |

### 1.3 Triangle

| Property  | Formula       | Variables              |
| --------- | ------------- | ---------------------- |
| Area      | A = ½bh       | b = base, h = height   |
| Perimeter | P = a + b + c | a, b, c = side lengths |

### 1.4 Parallelogram

| Property  | Formula      | Variables            |
| --------- | ------------ | -------------------- |
| Area      | A = bh       | b = base, h = height |
| Perimeter | P = 2(a + b) | a, b = side lengths  |

### 1.5 Rhombus

| Property  | Formula        | Variables          |
| --------- | -------------- | ------------------ |
| Area      | A = ½(d₁ × d₂) | d₁, d₂ = diagonals |
| Perimeter | P = 4s         | s = side length    |

### 1.6 Trapezium (Trapezoid)

| Property  | Formula           | Variables                         |
| --------- | ----------------- | --------------------------------- |
| Area      | A = ½(a + b)h     | a, b = parallel sides, h = height |
| Perimeter | P = a + b + c + d | a, b, c, d = side lengths         |

Note: For the trapezium, 'a' and 'b' are the lengths of the parallel sides.
